# AI 剧本杀设计大师

AI 一键生成完整谋杀之谜剧本，包含角色、线索、时间线、分幕剧本和主持人手册。

## 功能

- **剧本生成**: 输入人数/主题/难度 → AI 生成完整剧本（角色卡×N + 时间线 + 线索链 + 分幕剧本 + 主持人手册）
- **AI 修订**: 输入修改意见 → AI 局部修订
- **版本管理**: 自动保存编辑历史
- **剧本分享**: 生成分享卡片到微信群

## 技术栈

- Node.js + Express 5
- PostgreSQL (Supabase)
- DeepSeek AI
- WeChat 小程序

## 快速开始

```bash
npm install
cp .env.example .env
npm start              # http://localhost:8789
npm test               # 运行测试
```
